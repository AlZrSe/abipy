abiview.py¶

usage: abiview.py [-h] [--loglevel LOGLEVEL] [-V] [-v] [--seaborn]
                  {hist,abo,ebands,phbands,fields} ...

Named Arguments¶

--loglevel

Set the loglevel. Possible values: CRITICAL, ERROR (default), WARNING, INFO, DEBUG

Default: “ERROR”

-V, --version show program’s version number and exit
-v, --verbose

verbose, can be supplied multiple times to increase verbosity.

Default: 0

--seaborn

Use seaborn settings.

Default: False

subcommands¶

Valid subcommands

command

Possible choices: hist, abo, ebands, phbands, fields

sub-command help

Sub-commands:¶

hist¶

Visualize structural relaxation/molecular-dynamics run from data stored in the HIST.nc file. Requires mayavi.
abiview.py hist [-h] [-t] paths [paths ...]

Positional Arguments¶

paths List of files to analyze.

Named Arguments¶

-t, --trajectories
 

Plot trajectories.

Default: False

abo¶

Plot SCF iterations extracted from Abinit output file as well as timer data (if present)
abiview.py abo [-h] paths [paths ...]

Positional Arguments¶

paths List of files to analyze.

ebands¶

Animate electron bands. Accept any file with ElectronBands e.g. GSR.nc, WFK.nc, …

abiview.py ebands [-h] paths [paths ...]

Positional Arguments¶

paths List of files to analyze.

phbands¶

Animate phonon bands. Accept any file with PhononBands e.g. PHBST.nc, …

abiview.py phbands [-h] paths [paths ...]

Positional Arguments¶

paths List of files to analyze.

fields¶

Animate fields with Mayavi. Accept any file with density or potential …

abiview.py fields [-h] [--no-sort] paths [paths ...]

Positional Arguments¶

paths List of files to analyze.

Named Arguments¶

--no-sort

Disable automatic sorting of filepaths.

Default: False
