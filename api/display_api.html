abipy.display.utils.add_docstrings(*tuples)[source]¶

This decorator adds to the docstring the documentation for functions. When writing high-level API, it’s quite common to call thirdy-party functions with a restricted set of arguments while optional keyword arguments are collected in an optional dictionary.

The first item of the tuple contains the function (python object) wrapped by the code. The second item is list of strings with the name of the actual arguments passed to function.
