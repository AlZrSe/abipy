Changelog¶

0.3.0 2017-12-26

  • [Feature]: : Add Shankland-Koelling-Wood Fourier interpolation scheme. For the theoretical background see [ESC+69][KW86][PKA88][MS06].
  • [Feature]: Add several new options to abirun.py, abicomp.py and abistruct.py scripts.
  • [Feature]: Add robots for comparing/analyzing multiple files of the same type (abipy.dfpt.ddb.DdbRobot, abipy.electrons.gsr.GsrRobot …) Some of the robot capabilities are exposed via the abicomp.py and the abirun.py scripts.
  • [Feature]: Add interface with phononwebsite: abiview.py phbands out_PHBST.nc -web.
  • [Feature]: Change default value of shifts in abipy.abio.factories.g0w0_with_ppmodel_inputs().
  • [Feature]: Solve problem with visualize in jupyter notebooks (files should be produced in workdir)
  • [Feature]: Autodetect presence in abipy.dfpt.ddb.DdbFile of data required for the LO-TO splitting.
  • [Feature]: Add mpirunner_options and shell_runner_options to pymatgen.io.abinit.tasks.TaskManager.
  • [Feature]: Improve coverage
  • [Feature]: Add AbinitInput.set_spell_check to activate/deactivate spell-checker
  • [Feature]: abicheck.py accepts -with-flow options
  • [Feature]: Add abiview.py script for a quick visualization of results.
  • [Bug]: Fix bug in thermodinamical properties (zero-point energy was included twice)
  • [Bug]: Fix bug in in SigresFile.get_dataframe if QP states do not start with the same band index.
  • [Support]: Significant improvements to the documentation and the website: add AbiPy Gallery with matplotlib plots and Flow Gallery with AbiPy flows are now automatically generated.

0.2.0 2017-03-10

  • [Feature]: This is the first official release in which we have reached a relatively stable API and a well-defined interface with the netcdf files produced by Abinit. We recommend Abinit >= 8.0.8b, version 8.2.2 is required to analyze the electronic fatbands saved in the FATBANDS.nc file.
